Application Layer란?

Application Layer는 IPS(Internet Protocol Suite) 모델과 OSI(Open Systems Interconnection Reference Model) 모델의 1번째 계층이다.

Application Layer는 클라이언트의 요청을 전달하기 위해 서버가 이해할 수 있는 데이터로 변환하여 전송 계층으로 전달하는 역할을 수행한다. 클라이언트 측 응용 프로그램에서 서버 측 응용 프로그램과 통신하기 위해서는 Application Layer의 프로토콜을 사용해야한다.

Application Layer의 프로토콜로는 웹페이지 통신을 위한 HTTP, 파일을 전송하기 위한 FTP, 메일을 전송하기 위한 SMTP, 메일을 받기 위한 POP3, IMAP 등의 프로토콜이 있다.
